NodeMCU ESP8266




LoLin, üzerinde NodeMCU firmware yüklü ESP8266 WiFi modül barındıran bir geliştirme kartıdır. ESP8266 SDK'sı kullanılarak geliştirildiğinden, ekstradan bir mikrokontrolcüye ihtiyaç duymadan GPIO, PWM, IIC, 1-Wire ve ADC bağlantılarını destekler. Üzerinde CP2102 USB - Seri dönüştürücü entegre bulunur.

Teknik Özellikler

  • ESP8266 SDK tabanlı
  • eLua core (Lua 5.1.4)
  • lua-cjson destekli
  • spiffs dosya sistemi
  • json, file, timer, pwm, i2c, spi, 1-wire, net, mqtt, coap, gpio, wifi, adc, uart ve system api destekler
  • Ürün Boyutları: 2,4 x 4,8 cm
